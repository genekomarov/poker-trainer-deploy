# Изображения карт

PNG из репозитория [hayeah/playing-cards-assets](https://github.com/hayeah/playing-cards-assets) (MIT).

Исходные иллюстрации: [vector-playing-cards](https://code.google.com/p/vector-playing-cards/) (public domain).

Файлы: `2c.png` … `As.png`, `back.png` (нотация: ранг + масть).

Скачать заново:

```bash
npm run cards:download -w @poker-trainer/web
```
